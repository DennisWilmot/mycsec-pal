# MyCSECPal — Next.js

A Next.js App Router implementation of the MyCSECPal product prototype.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

- `/` — Landing page
- `/practice` — Practice dashboard
- `/paper-1` — Mathematics Paper 1
- `/paper-2` — Mathematics Paper 2
- `/results` — Results and review
- `/progress` — My Progress
- `/settings` — Profile and subscription

## Notes

- The interface is currently a front-end prototype with local component state.
- Replace mock data and navigation with your authentication, database, billing and AI grading services.
- Graphics are stored in `public/assets`.
